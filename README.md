# 3D Chess Game

A fully featured, beautiful 3D Chess game built with **Godot 4** and **C++ (GDExtension)**.

## ✨ Features
- **Full 3D Chess Experience**: High-quality 3D models with detailed 4K materials and textures.
- **Customizable Camera System**: Take the perfect shot! Click the in-game camera icon to adjust your angle and capture a snapshot of the board directly to your device.
- **Timed Matches**: Play with an intense chess clock, complete with tick-tock audio and a heartbeat effect when time runs low.
- **Dynamic Settings**: Change game resolution on the fly and access copyright/help information easily via the settings menu.
- **Polished UI**: Features a beautiful intro logo animation with wave shaders and a clean, minimalist game interface.

## 🛠️ Technology Stack
- **Engine**: Godot Engine 4.x
- **Logic**: C++ (via GDExtension for high performance) & GDScript (for UI and shaders)
